# Student_Management_System-C-
A C++ console-based Student Management System using OOP, File Handling, Searching, Sorting, and Data Persistence

## Overview

A console-based Student Management System developed in C++ using Object-Oriented Programming principles.

## Features

* Add Student
* Display Student Records
* Delete Student
* Delete All Students
* Search Student
* Sort by Age
* Sort by Name
* Sort by Grade
* Student Summary Report
* File Storage and Retrieval

## Technologies Used

* C++
* STL Vector
* STL Map
* File Handling
* OOP Concepts

## Learning Outcomes

* Object-Oriented Programming
* File Handling
* Data Structures
* Searching and Sorting Algorithms

## Future Enhancements

* Update Student Information
* Login Authentication
